# aegiscore-sdk examples

Six runnable example plugins, one per supported plugin kind. Each
directory is a buildable PyPI-style plugin package generated by
``aegiscore-sdk plugin new``.

| Directory | Kind | Entry-point group |
|-----------|------|-------------------|
| [`tool/`](tool/) | tool | `aegiscore.tools` |
| [`middleware/`](middleware/) | middleware | `aegiscore.middleware` |
| [`agent/`](agent/) | agent | `aegiscore.agents` |
| [`callback/`](callback/) | callback | `aegiscore.callbacks` |
| [`skill/`](skill/) | skill | `aegiscore.skills` |
| [`prompt/`](prompt/) | prompt | `aegiscore.prompts` |

## Try one

```bash
cd middleware
uv build
pip install dist/*.whl

python -c "from importlib.metadata import entry_points; print(entry_points(group='aegiscore.middleware'))"
```

## Regenerate

All six were generated by the SDK scaffolder. To regenerate (or to
scaffold your own), run from the workspace root:

```bash
uv run aegiscore-sdk plugin new \
    --kind=middleware \
    --name=my-plugin \
    --path=./my-plugin
```

See the umbrella spec at
[`docs/superpowers/specs/2026-05-23-core-framework-sdk-split-design.md`](../../../docs/superpowers/specs/2026-05-23-core-framework-sdk-split-design.md)

criteria these examples satisfy.
